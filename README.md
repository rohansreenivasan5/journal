## Getting Started

1. Clone or copy this template
2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) to see your app

## Development

- Edit `src/app/page.tsx` to modify the main page
- Add new pages in the `src/app/` directory
- Use `src/lib/` for utilities and shared code
- Styling is handled with Tailwind CSS

## Deployment

Deploy to Vercel with one click:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/webapp-template)

Or connect your GitHub repository to Vercel for automatic deployments.

## Customization

1. Update `package.json` with your project name
2. Modify `src/app/layout.tsx` metadata
3. Replace the landing page content in `src/app/page.tsx`
4. Add your own components and pages
